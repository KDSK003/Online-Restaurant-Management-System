# Online Restaurant Management System

## 📌 Project Overview
A web-based food ordering system developed using PHP and MySQL. 
Users can browse restaurants, add dishes to cart, and place orders online.

## 🛠 Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## 🚀 Features
- User Registration & Login
- Restaurant Listing
- Add to Cart
- Checkout System
- Order History
- Review System

## 🗂 Folder Structure
- assets → CSS, JS, Images
- database → SQL file
- screenshots → UI images

## 🧠 Conclusion
This project demonstrates full-stack development skills with database connectivity and session handling.
